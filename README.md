# gmail-to-pdf

Projeto com umas pá de coisa aí para conectar no Gmail via OAuth, listar e-mails recentes e gerar PDF com Puppeteer.


Tech Stack:

Frontend:
- HTML5
- CSS3
- JavaScript

Backend:
- Node.js
- Express.js

Authentication:
- Google OAuth 2.0

APIs:
- Gmail API

PDF Generation:
- Puppeteer

Infrastructure:
- Docker
- Railway (Cloud Hosting)

Deployment:
- GitHub Pages
- GitHub Actions (CI/CD)
